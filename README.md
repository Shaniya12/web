# web
Project Overview:

A web platform that empowers street vendors and small food businesses to directly source raw materials at wholesale prices, cutting out middlemen and reducing costs.

Key Features:

Vendor Dashboard: Street vendors can create accounts, browse suppliers, and place orders

Supplier Network: Verified wholesalers and producers list their inventory with competitive pricing

Price Comparison: Tools to compare prices across different suppliers

Bulk Order Discounts: Special pricing for collective orders from vendor groups

Delivery Coordination: Logistics support for efficient material delivery

Localized Experience: Available in multiple languages with region-specific suppliers

Goals:

Reduce raw material costs for street vendors by 20-40%

Create transparent supply chains

Support local economies

Digital empowerment of informal sector businesses


Help street vendors thrive by giving them direct access to affordable supplies! Join us in building this impactful platform.
